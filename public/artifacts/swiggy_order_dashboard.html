How to export your Swiggy order history

Follow these 3 steps to extract your orders and visualize them here.

1
Open Swiggy in your browser
Go to swiggy.com and make sure you are logged in to your account.
2
Open the browser console and run this script
Press F12 (or Cmd+Option+J on Mac) to open DevTools, go to the Console tab, paste the script below and hit Enter. It will fetch all your past orders page by page and automatically download a swiggy_orders.json file.
async function fetchAllSwiggyOrders() { const allOrders = []; let lastOrderId = ""; let page = 1; while (true) { console.log(`Fetching page ${page}...`); const res = await fetch(`/dapi/order/all?order_id=${lastOrderId}`, { headers: { "content-type": "application/json", "platform": "dweb", "__fetch_req__": "true" } }); const data = await res.json(); const orders = data?.data?.orders; if (!orders || orders.length === 0) { console.log("No more orders."); break; } allOrders.push(...orders); console.log(`Page ${page}: +${orders.length} orders | Total: ${allOrders.length}`); lastOrderId = orders[orders.length - 1].order_id; page++; await new Promise(r => setTimeout(r, 500)); } console.log(`✅ Done! ${allOrders.length} orders fetched.`); console.log(JSON.stringify(allOrders)); return allOrders; } const orders = await fetchAllSwiggyOrders();
3
Upload the downloaded JSON file below
Once the script finishes, a file called swiggy_orders.json will download automatically. Upload it using the panel below to see your full dashboard.
